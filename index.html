Pixel History

Pixel games has a long existing history since the 1970’s! This little tiny dot creates the first video game called Pong which was released in the 1970's. Pixel art have created so many pixel games including old-school Roleplaying Games and 2-D fighting games. Back in the day the first pixel art was created in embroidery but the word pixel was added ias a dictionary term by Fred C. Billingsley in the 1960's

To learn more about the history of pixels, check Wikipedia or this site when you click here!
